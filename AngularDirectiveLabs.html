Angular.js Exercises

Custom Directive Exercises

The following exercises will help you explore how to create custom Angular.js directives. The exercises will explore the following concepts:

  1. Restricting the type of marker with which a directive can be applied.
  2. Using inline and external templates with a directive.
  3. Transcluding the content of the element on which the directive is applied.
  4. Using the compile service to add new directives to link function’s element object.
  5. Isolating a directive’s scope to maximize re-usability.
  6. Creating a custom validator and exploring how to use other directives’ controllers.
  7. Explore directive priority and gain deeper understanding of the terminal option.
  8. Setting up observes and interacting with a directive’s attribute object in the link function.

Exercise Setup

For each exercise a new HTML file will need to be created. This can be done one of two ways. First, a new HTML file can be created with a script reference to Angular.js in the content of the HEAD element and the following code added to the content of the BODY element. 

<div ng-app="App">
    <div ng-controller="Ctrl">
    </div>
</div>
<script>

    angular.module("App", [])
        .controller("Ctrl", function() {

        });

</script>

The exercises will use the above code as their starting structure. The second way to create a new file is to simply copy the file from a previous exercise and remove the code from the previous exercise, thereby, resetting it the original code as shown above.

If you are unsure how to add a script reference to the top of an HTML file in the content of the HEAD section, please ask the instructor.

    template: "Observe Value: {{observeValue}}",
    link: function(scope, element, attrs) {
    }
}

Step 5. In the link function, setup an observe to observe the observe-exercise attribute to update the observeValue property on the scope.

Step 6. Initialize the Ctrl scope by adding the following code to the Ctrl controller function. If you are not familiar with JavaScript promises or how promises are used with the interval and timeout services, ask the instructor for further information. 

$scope.currentValue = 0;

var intervalPromise = $interval(function() {
    $scope.currentValue++;
}, 500);

$timeout(function() {
    $interval.cancel(intervalPromise);
}, 5000);

The interval service will increment current value approximately every 500 milliseconds. The interval will continue until being canceled by the callback function passed into the $timeout service approximately 5 seconds later.
